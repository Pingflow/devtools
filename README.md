# Dev Tools
Golang DevTools for [GoMicro](https://github.com/micro/go-micro)

## Dependencies
### Minimal
| Dependencies      | Version |
| ----------------: | :------ |
| docker            | 18.09.6 |
| docker-compose    | 1.17.1  |
  
## Development
| Dependencies | Version |
| -----------: | :------ |
| go           | 1.12.3  |

## Global variables
```env
DEVTOOLS_PATH=
``` 

## Dev commands
- build :
```sh
make build
```

- run :
```sh
make run
```

- generate release :
```sh
make realase
```
